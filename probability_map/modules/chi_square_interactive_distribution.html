Chi-Square Distribution $X \sim \chi^2(\nu)$
(degrees of freedom)
Probability Density Function
$$f(x) = \frac{1}{2^{\nu/2}\Gamma(\nu/2)} x^{\nu/2-1} e^{-x/2}$$
Key Properties
Support$x \geq 0$
Mean$E[X] = \nu$
Variance$\text{Var}[X] = 2\nu$
Mode$\max(0, \nu-2)$
Key Relationship
If $Z_i \sim N(0,1)$ i.i.d., then $\sum_{i=1}^{\nu} Z_i^2 \sim \chi^2(\nu)$
