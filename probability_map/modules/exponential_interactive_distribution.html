Exponential Distribution $X \sim \text{Exp}(\lambda)$
(rate)
Probability Density Function
$$f(x) = \lambda e^{-\lambda x}, \quad x \geq 0$$
Key Properties
Support$x \geq 0$
Mean$E[X] = 1/\lambda$
Variance$\text{Var}[X] = 1/\lambda^2$
CDF$F(x) = 1 - e^{-\lambda x}$
Memoryless Property
$P(X > s+t \mid X > s) = P(X > t)$
