Geometric Distribution $X \sim \text{Geom}(p)$
(success prob)
Probability Mass Function
$$P(X=k) = (1-p)^k p, \quad k = 0, 1, 2, \ldots$$
Key Properties
Support$k \in \{0, 1, 2, \ldots\}$
Mean$E[X] = \frac{1-p}{p}$
Variance$\text{Var}[X] = \frac{1-p}{p^2}$
Interpretation# failures before 1st success
Memoryless Property
$P(X > m+n \mid X > m) = P(X > n)$
