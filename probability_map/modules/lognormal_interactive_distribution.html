Lognormal Distribution $X \sim \text{Lognormal}(\mu, \sigma)$
(log-mean)
(log-sd)
Probability Density Function
$$f(x) = \frac{1}{x\sigma\sqrt{2\pi}} \exp\left(-\frac{(\ln x - \mu)^2}{2\sigma^2}\right)$$
Key Properties
Support$x > 0$
Mean$E[X] = e^{\mu + \sigma^2/2}$
Variance$(e^{\sigma^2}-1)e^{2\mu+\sigma^2}$
Median$e^\mu$
Key Relationship
If $Y \sim N(\mu, \sigma^2)$, then $X = e^Y \sim \text{Lognormal}(\mu, \sigma)$
