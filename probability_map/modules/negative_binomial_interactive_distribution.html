Negative Binomial Distribution $X \sim \text{NB}(r, p)$
(# of successes)
(success prob)
Probability Mass Function
$$P(X=k) = \binom{k+r-1}{k} (1-p)^k p^r$$
Key Properties
Support$k \in \{0, 1, 2, \ldots\}$
Mean$E[X] = \frac{r(1-p)}{p}$
Variance$\text{Var}[X] = \frac{r(1-p)}{p^2}$
Interpretation# failures before $r$-th success
