Geometric Distribution
(success probability)
Key facts
  • Support: k = 0, 1, 2, … (failures before first success)
  • PMF: P(X=k) = (1−p)ᵏ p
  • Mean: E[X] = (1−p)/p
  • Variance: Var[X] = (1−p)/p²
  • Memoryless property: P(X > m+n | X > m) = P(X > n)
